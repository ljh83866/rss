# 🌐 RSSWorker

**RSSWorker** 是一个基于 Cloudflare Worker 的 RSS 生成工具。它通过访问网页或调用 API，将内容转换为标准 RSS，可自部署使用。

## ⚙️ 功能原理
- 🌍 访问网页或调用各平台 API
- 📝 解析内容（文字、图片、链接等）
- 📡 生成 RSS feed

## ⛔ 已放弃的站点
- [Pixiv](https://www.pixiv.net) 理由：防火墙阻止WorkerIP访问，防爬可过
- [JavDB](https://javdb.com) 理由：防火墙有几率阻止WorkerIP访问，似乎无防爬
